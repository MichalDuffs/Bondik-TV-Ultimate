# 🏗️ Project Structure

This document explains the structure of **Bondik TV Ultimate** repository.

---

# 📂 Root Directory
Main project directory.

---

# 🎨 assets/

Contains all graphical resources.
Used for:

- README presentation
- Branding
- Documentation
- Application graphics

---

# 📺 playlists/

Contains IPTV playlists.
Organization:

🌍 Countries  
🎬 Categories  
📡 Providers  

---

# 📅 epg/

Contains Electronic Program Guide data.

Used for:

- TV schedules
- Program information
- Channel metadata

---

# 📚 docs/

Contains user and developer documentation.

Examples:

- Installation guides
- Player guides
- FAQ
- Troubleshooting

---

# 🛠️ tools/

Contains project maintenance tools.
Future automation:

- Playlist validation
- Updates
- Reports

---

# ⚙️ .github/

Contains GitHub configuration.

Includes:

- Workflows
- Issue templates
- Pull request templates
