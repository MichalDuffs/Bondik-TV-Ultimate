# ❓ Frequently Asked Questions (FAQ)

Welcome to the Bondik TV Ultimate FAQ.

Here you can find answers to common questions.

---

# 📺 What is Bondik TV Ultimate?

Bondik TV Ultimate is an open-source IPTV playlist project.

The goal is to provide organized playlists with:

- 🌍 Countries
- 🎬 Categories
- 📡 Providers
- 📅 EPG support

---

# 🔗 What is an M3U playlist?

M3U is a playlist format used by IPTV players.

It contains:

- Channel names
- Stream addresses
- Metadata
- Logos

Compatible players can load this file and display channels.

---

# 📅 What is EPG?

EPG means Electronic Program Guide.

It provides:

- Current program information
- Program schedules
- Channel descriptions

# 📺 Why does a channel sometimes not work?

Possible reasons:

- Stream provider changed the address.
- Channel is temporarily offline.
- Server is overloaded.
- Source is no longer available.

Try:

1. Refresh playlist.
2. Update playlist.
3. Try again later.

---

# 🔄 How often are playlists updated?

Updates depend on:

- Channel availability.
- Community reports.
- Automatic checks (when available).
