# 🚀 Installation Guide

Welcome to **Bondik TV Ultimate**.

This guide explains how to install and use IPTV playlists.

---

# 📋 Requirements

Before starting you need:

- IPTV compatible player
- Internet connection
- M3U playlist URL
- Optional EPG URL

---

# 🤖 Android / Android TV

## Installation steps

1. Install IPTV player from your app store.
2. Open the application.
3. Select:
   
      `Add Playlist`

      4. Choose:

         `M3U URL`

         5. Insert Bondik TV Ultimate playlist URL.
         6. Save and load channels.

         ---

         # 🪟 Windows

         ## VLC Player

         1. Open VLC.
         2. Select:
         3. Paste M3U URL.
         4. Press Play.

         ---

         # 🍎 iOS

         1. Install compatible IPTV player.
         2. Add M3U playlist.
         3. Add EPG if supported.

         ---

         # 📺 Smart TV

         Supported systems:

         - Android TV
         - Google TV
         - LG webOS
         - Samsung Tizen

         Steps:

         1. Install IPTV application.
         2. Add playlist.
         3. Load channels.

         ---

         # 📅 EPG Setup

         If your player supports EPG:

         1. Open settings.
         2. Add XMLTV URL.
         3. Refresh program guide.

         ---

         # 🔄 Updating Playlist

         To get the latest channels:

         1. Open playlist settings.
         2. Refresh playlist.
         3. Reload channels.
