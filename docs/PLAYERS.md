# 📺 Supported Players

## ⭐ Recommended Players

### IPTV Smarters Pro

✅ Easy setup  
✅ M3U support  
✅ EPG support  
✅ User friendly interface  

---

### VLC Media Player

✅ Free  
✅ Open Source  
✅ Available on many platforms

# 📺 Android TV

Recommended for:

- Smart TV
- TV Box
- NVIDIA Shield
- Fire TV

Features:

- Remote control support
- Large screen interface
- M3U playlists
- EPG support

# 🪟 Windows

## VLC

Steps:

1. Open VLC
2. Media → Open Network Stream
3. Insert M3U playlist URL
4. Play
