# 🛠️ Troubleshooting

Common problems and solutions.

---

# 📺 Channel does not work

Possible reasons:

- Stream is offline.
- Provider changed the link.
- Temporary outage.

Solution:

- Refresh playlist.
- Try again later.

---

# 📡 Buffering

Possible causes:

- Slow internet.
- Weak WiFi signal.
- Server overload.

Solutions:

- Use Ethernet connection.
- Restart router.
- Lower stream quality.

---

# 📅 EPG not showing

Check:

- Correct EPG URL.
- Player compatibility.
- Channel names matching.

---

# 🔄 Playlist update failed

Try:

- Remove old playlist.
- Add latest URL.
- Restart application.

---

# 🐞 Still a problem?

Create an issue on GitHub and include:

- Device
- Player name
- Error message
- Channel name
