# 🤝 Contributing to Bondik TV Ultimate

First of all, thank you for your interest in contributing to **Bondik TV Ultimate**!

Every contribution helps make the project better.

---

# 📌 Ways to Contribute

You can help by:

- 📺 Reporting broken IPTV streams
- 🌍 Adding new country playlists
- 🎬 Adding category playlists
- 📡 Adding provider playlists
- 📅 Improving EPG support
- 📝 Improving documentation
- 🐞 Reporting bugs
- 💡 Suggesting new ideas

---

# 📂 Playlist Rules

Before submitting a playlist:

- Verify that all streams work.
- Avoid duplicate channels.
- Use clear channel names.
- Keep playlists organized.
- Respect copyright and applicable laws.

---

# 📖 Documentation

Documentation should be:

- Clear
- Simple
- Up to date
- Written in Markdown

---

# 🛠️ Pull Requests

Please:

- Keep changes focused.
- Describe what was changed.
- Test everything before submitting.
- Follow the project structure.

---

# ❤️ Thank You

Every contribution is appreciated.

Together we can build one of the best open-source IPTV repositories.

🐾 Bondík thanks you for helping!
