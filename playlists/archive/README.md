# 🗄️ Playlist Archive

This directory contains archived playlists and removed entries.

---

# Purpose

Archive stores:

- Old playlist versions
- Removed channels
- Historical changes
- Testing versions

---

# Rules

Archived playlists are:

❌ Not used in ultimate.m3u  
❌ Not automatically generated  
✅ Kept for history and reference

---

# Why Archive?

Channels may disappear because:

- Stream stopped working
- Source changed
- Duplicate channel
- Better replacement available

---

# Status

Archive does not mean deleted.

It preserves project history.
